Repository ini berisi **kode-kode yang ditulis selama kelas berlangsung**.

Seiring berjalannya perkuliahan, praktik kecil mulai muncul di kelas. Daripada kode tersebut hilang begitu saja setelah kelas selesai, kami memutuskan untuk mengumpulkannya di sini agar bisa dipelajari kembali kapan saja.

Kode biasanya akan **diunggah setelah kelas selesai**, sehingga siapa pun bisa membuka kembali contoh yang dijelaskan saat pembelajaran.

Anggap saja repository ini seperti **catatan belajar berbentuk kode**.
